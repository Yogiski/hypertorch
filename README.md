# HyperTorch

Implementations of basic PyTorch NN modules that operate completely in the Lorentz model of hyperbolic space.
The Lorentz model of n-dimensional hyperbolic space is defined by the following manifold:
$$L = \{x = [x_0, x_1, ..., x_n] \in \mathbb{R}^{n+1} : ⟨x, x⟩_L = -1 , x_0 > 0 \}$$
